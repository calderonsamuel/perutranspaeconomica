# perutranspaeconomica

<!-- badges: start -->
<!-- badges: end -->

El objetivo de perutranspaeconomica es contar con una manera de obtener
datos del Portal de Transparencia Económica Perú desde R, con una
interfaz de programación consistente y clara.
